🔄 Sincronizar Baterías al Archivo Fuente

Esta herramienta sincroniza las baterías desde localStorage al archivo fuente para que se desplieguen correctamente.

📋 Instrucciones:
1. Asegúrate de estar en localhost (http://localhost:3000)
2. Haz clic en "Sincronizar Baterías"
3. Revisa los cambios con: git diff src/data/batteries.ts
4. Haz commit y deploy: git add src/data/batteries.ts && git commit -m "Update batteries" && vercel --prod --yes
