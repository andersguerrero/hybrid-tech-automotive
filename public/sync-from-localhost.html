🔄 Sincronizar Baterías desde Localhost

Esta herramienta lee las baterías desde localStorage y las sincroniza al archivo fuente.

📋 Instrucciones:
1. Asegúrate de estar en localhost (http://localhost:3000)
2. Haz clic en "Sincronizar desde LocalStorage"
3. El sistema actualizará automáticamente el archivo fuente
4. Luego ejecuta: git add src/data/batteries.ts && git commit -m "Sync from localhost" && vercel --prod --yes
